Configurar Google Calendar

Ingresa tu Client ID de OAuth 2.0. Al conectar se abrirá una ventana de Google — permite los popups si tu navegador los bloquea.

    Ve a console.cloud.google.com → crea un proyecto.
  2. 2
    Activa la Google Calendar API.
  3. 3
    Credenciales → OAuth 2.0 → tipo "Aplicación web".
  4. 4
    En "Orígenes JS autorizados" agrega https://aiirvizionz.github.io. Copia el Client ID.
